# 🌱 CropSentinel: AI-Based Crop Health & Flood Damage Detection using Deep Learning (VGG16, ResNet50) and OpenCV

CropSentinel is an AI-powered agricultural monitoring system designed to detect crop health and flood damage using deep learning and computer vision techniques.

The system utilizes pre-trained convolutional neural network models such as VGG16 and ResNet50 along with OpenCV to perform real-time plant analysis through a webcam. It can instantly classify crops as healthy or diseased, enabling early intervention and reducing agricultural losses.

---

## 🚀 Features

- 🌿 Real-time plant health detection using webcam
- 🤖 Deep Learning models: VGG16 & ResNet50 (Transfer Learning)
- 📸 Image processing with OpenCV
- ⚡ Instant classification: Healthy vs Diseased crops
- 🌊 Flood damage detection (remote sensing concept)
- 📊 Performance comparison of models

---

## 🛠️ Tech Stack

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Pre-trained CNN Models (VGG16, ResNet50)

---

## 📌 How It Works

1. Capture live video using webcam
2. Extract frames using OpenCV
3. Preprocess image for model input
4. Predict using trained deep learning model
5. Display result: Healthy or Diseased
